============
Installation
============


Stable release
--------------

To install Swc Parser, run this command in your terminal:

.. code-block:: console

    $ pip install swc_parser

This is the preferred method to install Swc Parser, as it will always install the most recent stable release.

If you don't have `pip`_ installed, this `Python installation guide`_ can guide
you through the process.

.. _pip: https://pip.pypa.io
.. _Python installation guide: http://docs.python-guide.org/en/latest/starting/installation/


From sources
------------

The sources for Swc Parser can be downloaded from the `Github repo`_.

You can either clone the public repository:

.. code-block:: console

    $ git clone git://github.com/pattonw/swc_parser

Or download the `tarball`_:

.. code-block:: console

    $ curl  -OL https://github.com/pattonw/swc_parser/tarball/master

Once you have a copy of the source, you can install it with:

.. code-block:: console

    $ python setup.py install


.. _Github repo: https://github.com/pattonw/swc_parser
.. _tarball: https://github.com/pattonw/swc_parser/tarball/master
